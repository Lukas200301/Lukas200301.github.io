Features

Real-Time Dashboard

  • Smooth vector graphs showing 60-second history
  • Detailed memory breakdown (Used/Free/Cached/Swap)

  • Network interface details and disk usage

Docker Manager

Complete Docker container management at your fingertips:

  • Live container log streaming with auto-scroll
  • Container details: images, ports, creation dates
  • Quick access to container management

Service Manager

Full control over systemd services with intuitive management:

  • List all services (Active/Inactive/Failed)
  • Start, Stop, Restart services instantly
  • Enable or Disable services at boot
  • Search and filter by service name
  • Real-time service status monitoring

Keep your Raspberry Pi up-to-date with ease:

  • System information overview (OS, kernel, architecture)
  • Check for available package updates
  • Per-package or bulk system upgrade
  • Real-time streaming upgrade progress logs
  • Safe and controlled update process

Advanced Terminal

Professional-grade SSH terminal with enhanced productivity:

  • Full xterm-256color compatible shell
  • Custom accessory keyboard (Esc, Tab, Ctrl, Alt, Arrows)
  • Command history support for quick access
  • Copy/paste functionality
  • Multi-session support

SFTP File Explorer

Seamless file management with powerful features:

  • Browse remote directories intuitively
  • Upload/Download with progress tracking
  • File previews with syntax highlighting
  • Rename, delete, create folders
  • Change file permissions
  • Multi-file operations

Package Manager

Comprehensive package management interface:

  • Search for packages in repositories
  • Install and remove packages easily
  • View detailed package information
  • Dependency management
  • Streaming operation logs

System Logs

Real-time log monitoring with powerful filtering:

  • Live log streaming via journalctl
  • Filter by log level (Error/Warning/Info)
  • Service-specific log filtering
  • Search and highlight capabilities
  • Export logs for analysis

Connection Manager

Reliable multi-device connection management:

  • Save multiple Pi connections securely
  • Favorite connections for quick access
  • Auto-reconnect with configurable retry logic
  • Connection history and statistics
  • One-tap device switching

About This Project

Project Overview

A high-performance, real-time monitoring and control application for Raspberry Pi featuring a beautiful cyberpunk-inspired glassmorphic UI. Built with modern technologies, it offers blazing-fast updates, comprehensive system management, and secure communication.

High-Performance Architecture

Revolutionary gRPC-based architecture for maximum speed:

  • Flutter 3.10+: Beautiful cross-platform UI
  • Go 1.25+ Backend: Zero-dependency agent
  • gRPC + Protobuf: High-speed binary streaming
  • SSH Tunnel: All traffic encrypted
  • Auto-Deploy: Agent installs in ~5 seconds

Supported Devices

Works with all Raspberry Pi models:

  • Pi Zero/Zero W: ARMv6 support
  • Pi 2/3: ARMv7 architecture
  • Pi 3/4/5 64-bit: ARM64 optimized
  • Smart Detection: Automatic architecture selection

Beautiful Design

Cyberpunk/Sci-Fi Glassmorphism Theme features:

  • AMOLED Black: #000000 for battery efficiency
  • Frosted Glass: Cards with blur effects
  • Vibrant Accents: Electric Indigo & Teal
  • Smooth Animations: Fluid transitions

Getting Started

Prerequisites

  • Flutter SDK 3.10+ - Run flutter doctor to verify
  • Go 1.25+ (for building agent) - Run go version to check
  • Protocol Buffers Compiler (protoc) - Required for code generation
  • A Raspberry Pi with SSH enabled (port 22)
  • Network connectivity to your Raspberry Pi
Install Protocol Buffers Compiler 
# Windows (using Chocolatey)
choco install protobuf

# macOS (using Homebrew)
brew install protobuf

# Linux (Ubuntu/Debian)
sudo apt-get install protobuf-compiler

# Verify installation
protoc --version

Installation

1. Clone Repository 
git clone https://github.com/Lukas200301/RaspberryPi-Control.git
cd RaspberryPi-Control
2. Build Everything (Protobuf + Agent Binaries) 
# Windows PowerShell
.\build_all.ps1

# This generates:
# - Dart protobuf code in lib/generated/
# - Go protobuf code in agent/proto/
# - Cross-compiled agent binaries for ARM6, ARM7, ARM64
3. Install Dependencies & Run 
# Install Flutter dependencies
flutter pub get

# Run the app
flutter run

# Or build for production:
# Android
flutter build apk --release

# Windows
flutter build windows --release

First Connection

Auto-Installation: When you first connect to a Pi, the app automatically deploys the Go agent (~5 seconds). Subsequent connections are instant!
Security: All communication flows through an SSH tunnel. No firewall configuration needed. Passwords are encrypted in your device keychain.

Technical Details

gRPC Architecture

High-Performance Communication:

  • gRPC with Protocol Buffers for binary streaming
  • 500ms update intervals for real-time monitoring
  • Bidirectional streaming for live logs
  • Compact binary format reduces bandwidth
  • Type-safe API contracts via protobuf

Go Backend Agent

Zero-Dependency Performance:

  • Single binary deployment (~10MB)
  • No external dependencies required
  • Cross-compiled for ARMv6, ARMv7, and ARM64
  • Auto-deployed via SSH in ~5 seconds
  • Minimal resource footprint on Pi

Security & Privacy

Enterprise-Grade Protection:

  • All traffic encrypted via SSH tunnel
  • No open ports required on Pi
  • Passwords stored in device keychain
  • Agent only listens on localhost
  • No data collection or telemetry

Smart Features

Intelligent Operation:

  • Automatic Pi architecture detection
  • Version checking with auto-update
  • Connection resilience with retry logic
  • Background service support (mobile)
  • Graceful error handling and recovery
