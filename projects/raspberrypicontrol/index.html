Raspberry Pi Control

A comprehensive application for controlling your Raspberry Pi from Android and Windows devices. This app allows you to monitor system information, execute commands, and manage your Raspberry Pi remotely.

About This Project

Project Overview

This project provides a cross-platform solution for monitoring and controlling Raspberry Pi devices remotely. This guide offers information about the project's features, structure, and how to use the application effectively.

Project Architecture

The project is built using Flutter/Dart for cross-platform compatibility and consists of:

  • Mobile Applications: For portable devices
  • Desktop Client: For computer systems
  • Communication Layer: Handles secure connections and data transfer

Development Environment

For developers interested in contributing:

  • Clone the repository from the GitHub page
  • Install the necessary development SDKs
  • Install all project dependencies
  • Use a modern IDE with appropriate plugins

Testing Environment

For testing purposes:

  • Mobile: Use appropriate development targets
  • Desktop: Build for supported platforms
  • Hardware: Requires a properly configured target device

Getting Started

Prerequisites

  • A target Raspberry Pi device with SSH enabled
  • Network connectivity between your device and the Raspberry Pi
  • For developers: Required development SDKs

Installation

Clone Repository 
# Android Installation
git clone https://github.com/Lukas200301/RaspberryPi-Control.git
cd RaspberryPi-Control

# Or download directly from GitHub releases
# https://github.com/Lukas200301/RaspberryPi-Control/releases
Build & Install 
# For Android
flutter build apk --release
adb install build/app/outputs/flutter-apk/app-release.apk

# For Windows
flutter build windows --release
# Navigate to build/windows/runner/Release
Launch Application 
# For Android - Find the app in your app drawer
# For Windows - Run the executable from the Release folder
RaspberryPi-Control.exe

Important Notes

System Requirements: The target Raspberry Pi device must be configured with supported locales for all monitoring features to work correctly.
Device Settings: To maintain connections in the background on mobile devices, you may need to adjust battery optimization settings.
