NetW1re

Advanced Network Monitoring and Control Tool. A modern network management tool that allows you to monitor, analyze, and control the traffic of devices on your local network. Built with Avalonia UI and following the MVVM pattern, NetW1re provides a sleek, cross-platform experience for network administrators and power users.

Troubleshooting

Common issues and their solutions to help you get the most out of NetW1re

Device Not Appearing

Problem

Devices on your network are not showing up in the NetW1re interface

Solutions
  • Ensure the device is connected to the same network subnet
  • Wait a few minutes for automatic discovery to complete
  • Use the 'Refresh' button to force a new network scan
  • Check if the device is in sleep or power-saving mode
  • Verify firewall settings aren't blocking discovery

Actions Not Working

Problem

Block, redirect, or bandwidth limit actions are not being applied to target devices

Solutions
  • Verify NetW1re is running with administrator privileges
  • Check network adapter compatibility with packet injection
  • Ensure target device is still online and responsive
  • Restart NetW1re if issues persist
  • Disable conflicting network management software

Performance Issues

Problem

NetW1re is consuming high CPU or memory resources

Solutions
  • Reduce network scan frequency in settings
  • Close packet monitoring when not actively needed
  • Limit the number of active traffic control rules
  • Check available system memory and close other applications
  • Update to the latest version for performance improvements
