Raspberry Pi Control Guide

Dart 3 View on GitHub

A comprehensive application for controlling your Raspberry Pi from Android and Windows devices. This app allows you to monitor system information, execute commands, and manage your Raspberry Pi remotely.

📱 Features

Core Functionality

Complete control center for Raspberry Pi devices with comprehensive management tools:

  • SSH connectivity to devices
  • Advanced terminal interface for command execution
  • Comprehensive system monitoring
  • Multiple connection management
  • Cross-device file transfer
  • Service management
  • Customizable theme support

System Monitoring

Real-time insights into your Raspberry Pi's performance with intuitive visualization tools:

  • System log viewing
  • Network diagnostics tools
  • Process monitoring and management
  • Network connection tracking
  • Fully customizable metrics dashboard

Connection Features

Reliable connection management with enhanced stability and security features:

  • Background connection maintenance
  • Intelligent reconnection handling
  • Advanced connection configurations
  • Security timeout options
  • Network optimization settings

Terminal Experience

Powerful command-line interface with productivity enhancements for efficient device control:

  • Feature-rich terminal environment
  • Command history management
  • Text manipulation tools
  • Support for complex shell operations
  • Optimized output handling

File Management

Seamless file handling between your devices and Raspberry Pi with comprehensive tools:

  • Intuitive remote file browsing
  • Bidirectional file transfer system
  • File visibility options
  • Configurable file operations
  • Custom handling preferences
  • Robust error management

Cross-Platform Support

Available for multiple platforms with a consistent user experience:

  • Designed for various device form factors
  • Mobile support
  • Desktop support

🏁 Getting Started

Prerequisites

  • A target Raspberry Pi device with SSH enabled
  • Network connectivity between your device and the Raspberry Pi
  • For developers: Required development SDKs

Installation

  1. Download the application: 
    # Android Installation
git clone https://github.com/Lukas200301/RaspberryPi-Control.git
cd RaspberryPi-Control

# Or download directly from GitHub releases
# https://github.com/Lukas200301/RaspberryPi-Control/releases
  2. Install on your device: 
    # For Android
flutter build apk --release
adb install build/app/outputs/flutter-apk/app-release.apk

# For Windows
flutter build windows --release
# Navigate to build/windows/runner/Release
  3. Launch the application: 
    # For Android - Find the app in your app drawer
# For Windows - Run the executable from the Release folder
RaspberryPi-Control.exe

Important Notes

⚠️ System Requirements: The target Raspberry Pi device must be configured with supported locales for all monitoring features to work correctly.

⚠️ Device Settings: To maintain connections in the background on mobile devices, you may need to adjust battery optimization settings.

Background Operation

The application maintains connections even when minimized. System notifications indicate active status. For optimal performance:

  1. Enable notification permissions
  2. Configure appropriate power management settings
